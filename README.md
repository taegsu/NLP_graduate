# NLP_graduate

## 1. Git clone
```
> git clone https://github.com/taegsu/NLP_graduate.git
```

## 2. Install virtualenv & active
```
> virtualenv .venv
> source .venb/bin/activate
```

## 3. Install package
```
> pip install -r requirements.txt
```

## 4. Run server
```
> python manage.py runserver
```
